## HTML DOM
---
将 HTML 文档表达为树结构, 定义了访问和操作 HTML 文档的标准方法

> HTML DOM <=> 映射( html 文本文档 -> HTML DOM 对象和方法 )

## 虚拟 DOM
---

用普通 js 对象来描述的 DOM 结构，用于操作，时机成熟时再据此一次性改变 DOM

> 双射( 虚拟 DOM <-> HTML DOM )

### 为什么

直接操作 DOM 太卡
