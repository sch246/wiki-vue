# 搜索


我的关键词收藏夹

善用搜索引擎

冷门的选项有的是在个人网站或者b站等评论区找到的，因此搜索相关的视频然后找评论区亦是一种方法

百度搜不到想要的东西可以换个搜索引擎，比如bing或者google，可能百度两小时bing一分钟)

    - 文献: sci-hub, sci-hub中文社区, web of science, 谷歌学术

    - 图片: PicGo, ShareX, Google Photos
