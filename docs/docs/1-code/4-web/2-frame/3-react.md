# React.js


## 什么是 React.js？
> React 是用于构建用户界面的 JavaScript 库。
>
> 自发布以来，React 已成为前端应用程序最常用的库之一。
>
> React 组件因其声明性而易于使用，它用于轻松设计交互式用户界面、移动应用程序、单页应用程序等等。
