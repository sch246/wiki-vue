看完了八卦，先把它放到一边

朴素来讲，如果我们想打印出所有的组合，应该怎么做呢

先给阴阳两爻排个序，然后写个python脚本

```py
# 这里令阳爻为0，阴爻为1，其实反过来也一样
# 写成01只是方便辨认
阳='0'
阴='1'
for a in [阳,阴]:
    for b in [阳,阴]:
        for c in [阳,阴]:
            print(a+b+c)
```

它会输出这样的结果

```
000
001
010
011
100
101
110
111
```

然后你就会发现这就是乾兑离震巽坎艮坤的顺序

并且把这个转成二进制的话，就是 0,1,2,3,4,5,6,7，其中在二进制中位数越高，数字越大的爻，越在下面

当然你也可以试试反过来从下面往上面读，或者把01互换

依旧是这个顺序，并且也可以作为二进制排列

接下来看方位

按照这个顺序，它是这样排列的

![从上往下，先左后右](https://s2.loli.net/2023/02/02/WHhc1r7lqgjM35O.png)

方向是从上往下，先左后右的

并且，所有相对的爻都是相反的，因此所有相对的卦也都是相反的

你可能注意到了这在之前就有体现——阴阳互换或者倒过来读都不影响顺序

这是因为增加和减小数字只是从低位变化，向高位进1或者退1

增加或者减少数字，这2个动作本身是对等的

*一个极端的例子就是二进制中的其中一位，不论是加一还是减一，它总是从一个状态跳到另一个状态：如果是0就变成1，如果是1就变成0*

*超范围的运算总是影响更高一位：1+1则进1，0-1则退1，传递给更高位的也是对等的*

逆着顺序走一遍八卦，它是中心对称的

![](https://s2.loli.net/2023/02/02/4BNi1pW9LflUK2P.png)

于是八卦也可以理解为，乾坤两卦从两个端点同时出发，逆时针旋转而成，而在阴阳交汇处连接

按照同样的道理，到了7之后再增加就溢出到0了，因此7应该可以再指向0

```py
            for d in [阳,阴]:
                for e in [阳,阴]:
                    for f in [阳,阴]:

```

好长

简化一下，把循环换成递归

```py
阳='0'
阴='1'

def 二分(n, 卦=''):
    if n==0:
        print(卦)
    else:
        二分(n-1, 卦+阳)
        二分(n-1, 卦+阴)
# 这个同样可以生成八卦，把6换成3就行
二分(6)
```

其中的方阵就是从下至上(从北到南)，从右至左(从西到东)一行行排列的

如果分成上下2个八卦的话就能组成一个离散的平面直角坐标系

换成数字大小的话可以发现从西到东从北到南都是严格递减的(其中从北到南减小得比从西到东快8倍)

接下来只看外面的圈

*可以把外面的圆图也看成是两个八卦的卦拼在一起变成的：上面的那个卦过了一遍后，下面的卦就也变一下（进位），不过这样想的话，六十四卦就变成两重八卦了，没什么意思*

与先天八卦一样，所有相对的爻也都是相反的，因此所有相对的卦也都是相反的

所以同样可以理解成是乾坤两卦从两个端点逆时针而生成的六十四卦，而在阴阳之交的地方汇合

*有意思的是，阴阳之交的两卦也保持着完全的相反，并且这是所有可观测进位中的最高位进位/退位，比它更大的只有乾卦与坤卦的变换，那涉及的是范围外的位*

它把八卦的3位(爻)拓展到了6位(爻)，因此有了更大的观察窗口

另外，除了第一爻外，所有其它爻都是左右对称相同的，这同样是因为增减的地位是等价的，对于阴的减就是对于阳的增，反之亦然，所以负负得正，左右就对称了
