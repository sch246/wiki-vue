## 什么是道？

总之我先摆个[道德经](https://www.zhonghuadiancang.com/xuanxuewushu/daodejing/)在这里（（

我个人认为「道」就是：对「现象」不断问「为什么」，然后取极限的结果

$$\lim_{n \to \infty} (为什么^{n})(现象) = 道$$

怎么理解呢？

### 现象

凡是能察觉的都是「现象」，或者说叫「感受」

我们可以认为一切「现象」都由一定量的信息来认知

我们不关心它是什么样，怎么分的，也不关心有没有最小单位

我们只知道，当不同的「现象」放一起比较时，其中相同的部分就会凸显出来

这种「相同的部分」就是「共性」，而把一堆「现象」聚集到一起，找出「共性」的过程，就叫作「抽象」

人天生能抽象，我们思考的基本单位就是「共性」，或者说，「概念」。

比较的对象不同，抽象的方法不同，那么得到的「共性」也不同

通过各方面的比较，我们可以得到足够的「共性」来重新描述某个「现象」，这叫作「具象」

当然可以有不止一种方式描述同一个「现象」，就像同一个场景有不同的写法一样

当这个「现象」是「共性」时，这就是用概念重新定义概念

语言文字就是「共性」的一种载体，交流的过程就是是相互编码解码的过程，只要「共性」系统不差太远，那么便可以还原##人与人的悲欢并不相通.jpg##

这里其实有两层加密解密：

```flow
st=>start: 现象/感受
a=>operation: 共性/概念
w=>inputoutput: 文字/语言/..
www=>inputoutput: 文字/语言/..
b=>operation: 共性/概念
e=>end: 现象/感受

st(right)->a(right)->w->www(right)->b(right)->e
```

##是不是感觉绕了远路？抽象出的「共性」或者对应到的语言总会有偏差，所以咱直接感受「现象」不就好了嘛，我寻思也许这就是「忘言」##

### 为什么

啊，扯远了

人为什么问「为什么」呢？

有2个缘故

- 怀疑不靠谱

因为感觉到了某个「认知」并不靠谱，故欲「证明」其原因

当其原因被找到且证实后，不但可以放心使用这个「认知」，还可以用其原因生成类似的「认知」

::: tip 啥是「认知」？

可以应用的「共性」，通过对有时间顺序的「现象」进行抽象得到

就像是函数一样，不过以「现象/感受」的一部分作为输入，且输入一般可以主动干涉，据此人可以进行「想象/推理」

这让我想到了 lambda 演算，，

**「证明」是什么？**

在一段不能直接判断的「因果」上面进行「标记」，使得每两个「标记」之间，「显然可得」

本来成立的「因果」不需要「证明」，它唯一的作用就是给人看而已

:::

- 懒得记

与其说是寻求原因，倒不如说是在分解「认知」，使得它能被一组更加基本(更加稳定更加普适)的「认知」推出——成为某种可被具象的「现象」

当能被自然推出时，这个「认知」就可以忘掉了

虽然每个分解都会产生更多的「为什么」，但是例子越多，越是分解，作为基底的「认知」就越少越简单——这是一种抽象

通过专门记忆这部分「认知」，对于新的「认知」进行理解时往往只需要把原有「认知」具象化就行了，学习理解和记忆也就越快

这部分「认知」就是一个人的「观念」，而求道就是在不断重构「观念」

于是为什么求道也解释清楚了##，果然懒才是人类进步的阶梯##
